# setup関係
1. .envファイルを作成
```.env
# supabaseの設定から参照してください
VITE_SUPABASE_URL="Setting/Data API/Project URL/URL"
VITE_SUPABASE_ANON_KEY="API Keys/Legacy anon, service_role API keys/anon public"
VITE_PUBLIC="public"
```

2. npm インストール
```
cd frontend
npm install
```

3. **作動確認**
```
npm run dev
```
